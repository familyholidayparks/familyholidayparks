How the ranking works

Each park's Family Score weighs entertainment (20), nature (20), value (15), cleanliness (15), site size (10), guest sentiment (10) and location (10). Scores update as new review data flows in, and the Trend column tracks each park's movement between updates. Rankings are national: filtering by state keeps each park's Australia-wide rank so you always know where it truly stands.
